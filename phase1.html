🏃‍♂️ 1. Aerobic & Anaerobic Conditioning

a) Running Intervals (30 Minutes)

Preferable to cycling to avoid leg bulk. Perform in this structure:

  • Warm-Up: 5 mins slow steady pace.
  • Interval Work: 5 reps of 1 min maximum effort (90-95%) followed by 1 min slow recovery jog.
  • Cool-Down: 5 mins slow jog.
  • Progression: Make intervals 10 seconds longer each session (up to 2 mins work / 1 min off x 5).

b) Burpees (10 Minutes)

Goal: Complete 4 sets with 1 minute rest in between.

  • Start Load: 8 reps per set on your first session.
  • Progression: Add 1 more rep per set each session (e.g., reaching 18 reps/set by session 10).

c) Flexibility (15 Minutes)

Hold stretches for 20 seconds, release for 10 seconds, and repeat for 20 seconds. Target muscles:

Hamstrings • Quadriceps • Calf • Groin • Latissimus Dorsi • Shoulders • Chest • Forearm Flexors & Extensors

🧗 2. Low-Intensity Endurance

Pick option A or B (do not do both in one session). Make sure to warm up first.

Option A: Routes 4 x 4s

Select four different routes of the same grade that you can climb consecutively without resting. Lower off and move to the next route as quickly as possible. Repeat 4 times with rest times equal to total climbing time.

Grade suggestion: 2 notches below onsight (beginners/intermediates) or 4 below (advanced/elite).

Target Wall Angles:
  • Beginner/Low-Intermediate: Vertical
  • Intermediate: 5-10° Overhanging
  • Advanced: 10-20° Overhanging
  • Elite: 20-30° Overhanging

Option B: Random Bouldering (Traversing / Random holds)

Find an easy/quiet wall. Climb continuously for 10 minutes, using random holds going up, down, diagonally, and traversing. Goal is a continuous moderate forearm pump. If too pumped, find a resting stance and shake out before continuing. Rest 10 minutes and repeat for 4 total sets.

🧗‍♂️ 3. Bouldering Volume Pyramids

Climb problems in ascending then descending order of difficulty. Rest 1 minute between easy problems, and 2 minutes between harder grades. Try to add one more problem per grade each session.

Beginner / Low Intermediate Pyramid

V0 ➔ V1 ➔ V2 ➔ V1 ➔ V0

5x V0 • 5x V1 • 5x V2 • 5x V1 • 5x V0

Intermediate Pyramid

V0 ➔ V1 ➔ V2 ➔ V3 ➔ V4 ➔ V3 ➔ V2 ➔ V1

4x V0 • 4x V1 • 4x V2 • 4x V3 • 4x V4 • 4x V3 • 4x V2 • 4x V1

Advanced Pyramid

V1 ➔ V2 ➔ V3 ➔ V4 ➔ V5 ➔ V4 ➔ V3 ➔ V2

4x V1 • 4x V2 • 4x V3 • 4x V4 • 4x V5 • 4x V4 • 4x V3 • 4x V2

Elite Pyramid

V2 ➔ V3 ➔ V4 ➔ V5 ➔ V6 ➔ V7 ➔ V6 ➔ V5 ➔ V4 ➔ V3 ➔ V2

3x of each grade ascending and descending (Total: 33 climbs)

💪 4. Antagonists & Core

Antagonist Movements

Perform 3 sets of 20 repetitions. Rest 2 minutes between sets. Stop just before failure (failure is optional on the final set only).

  • Push-Ups: Keep spine straight (kneel if needed).
  • Reverse Wrist Curls: Using light comfortable weights.
  • Finger Extensions: Wrap a thick rubber band around fingers and spread wide.

Core Circuit

  • Extreme Plank: 10 reps x 3 sets (2m rest). Add 1 rep per session.
  • Iron Cross Plank: 10 reps x 3 sets (2m rest). Plank with arms and legs spread wide. Add 1 rep per session.
  • Leg Paddles: 50 reps x 3 sets (2m rest). Lie on back, crunch up halfway with hands on temples. Extend legs straight out, hover above floor, and flutter/paddle. Add 5 reps per session.
