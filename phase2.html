Climbing Training Plan

Phase 2: Low-Intensity Endurance (6 Weeks)

🧗 1. Low-Intensity Endurance

Adapt this program to your local facility using the following variations. Do not complete more than one option in a single session. Alternating options is ideal.

Option A: Routes - 5 x 3s (3-in-a-row)

Select 3 different routes of the same grade that you can climb consecutively.

  • Grade: At least 2-3 grades below onsight (Beginner/Intermediate) or 4 below (Advanced/Elite).
  • Transition: Lower off and switch to the next route as rapidly and safely as possible.
  • Structure: Climbing all 3 routes constitutes 1 set. Do 5 total sets with rest matching climbing duration.
  • Progression: Bump up the route grading by one letter sub-grade in subsequent sessions.

Option B: Routes - Up-down-ups

Climb up a route, down-climb a route, and then scale the first route a second time without resting.

  • Grades (Beginner/Intermediate): Up-climb is 2 grades below onsight; down-climb is 3 grades below onsight.
  • Grades (Advanced/Elite): Up-climb is 3 grades below onsight; down-climb is 5 grades below onsight.
  • Structure: Do this sequence 5 times, matching rest time to climbing time.
  • Progression: In subsequent sessions, step up the difficulty of the up-climb by 1 letter grade first, and then the down-climb by 1 letter grade next.

Option C: Bouldering Wall - Random Climbing

On a quiet section of wall, climb continually for 10 minutes, choosing holds at random (up, down, traverse, diagonally). Avoid peak fatigue: rest in specialized stances and shake-out to preserve a moderate forearm pump.

Recommended Session Timing Intervals:

5m on / 5m off ➔ 10m on / 10m off ➔ 15m on / 15m off ➔ 10m on / 10m off ➔ 5m on

Optimum Wall Angle Guidelines:
  • Beginner / Low-Intermediate: Vertical
  • Intermediate: 5 to 10 degrees overhanging
  • Advanced: 10 to 20 degrees overhanging
  • Elite: 20 to 30 degrees overhanging

Finisher

Complete all endurance workouts with sets of pull-ups and straight-leg raises directly to structural failure:

  • Beginner: 2 sets
  • Intermediate: 3 sets
  • Advanced / Elite: 5 sets

🧗‍♂️ 2. Bouldering Volume Pyramids

Scale climbs in a logical difficulty pyramid. Focus on climbing mileage rather than working limit projects. Rest 1 minute between climbs at the first 2 grades, 2 minutes at levels 3-4, and 3-4 minutes on the peak grade. Attempt to increase peak or sub-peak grade reps by 1 each session.

Beginner / Low Intermediate Pyramid

V0 ➔ V1 ➔ V2 ➔ V3 ➔ V2 ➔ V1

4x V0 • 3x V1 • 3x V2 • 1x V3 • 3x V2 • 3x V1

Intermediate Pyramid

V0 ➔ V1 ➔ V2 ➔ V3 ➔ V4 ➔ V5 ➔ V4 ➔ V3

4x V0 • 3x V1 • 2x V2 • 2x V3 • 2x V4 • 1x V5 • 2x V4 • 2x V3

Advanced Pyramid

V1 ➔ V2 ➔ V3 ➔ V4 ➔ V5 ➔ V6 ➔ V5 ➔ V4 ➔ V3 ➔ V2

4x V1 • 3x V2 • 2x V3 • 2x V4 • 2x V5 • 1x V6 • 2x V5 • 2x V4 • 2x V3 • 2x V2

Elite Pyramid

V2 ➔ V3 ➔ V4 ➔ V5 ➔ V6 ➔ V7 ➔ V8 ➔ V7 ➔ V6 ➔ V5 ➔ V4 ➔ V3

3x V2 • 3x V3 • 2x V4 • 2x V5 • 2x V6 • 2x V7 • 1x V8 • 2x V7 • 2x V6 • 2x V5 • 2x V4 • 2x V3

🏃‍♂️ 3. Conditioning & Flexibility

Maintained from Phase 1 to consolidate general physical capacity.

a) Running Intervals (30 Minutes)

  • Scale to include 3 to 4 high-output intervals.
  • Maintain steady aerobic speeds elsewhere to promote circulatory recovery.

b) Conditioning Circuit (10 Minutes)

Burpees or rope skipping options: [ 1 min on / 1 min off ] x 5 sets.

c) Flexibility (15 Minutes)

Hold target stretches for 20 seconds. Release for 10. Repeat twice per position.

💪 4. Antagonists & Core

Antagonist Movements

Perform 3 sets of 20 repetitions. Rest 2 minutes between sets. Stop just before failure (failure is optional on final set).

  • Push-Ups: Keep spine straight (kneel if required).
  • Reverse Wrist Curls: Using light comfortable weights.
  • Finger Extensions: Wrap thick rubber bands around fingers and spread wide.

Core Circuit

  • Extreme Plank: 10 reps x 3 sets (2m rest). Add 1 rep per session.
  • Iron Cross Plank: 10 reps x 3 sets (2m rest). Plank with arms and legs spread wide. Add 1 rep per session.
  • Leg Paddles: 50 reps x 3 sets (2m rest). Lie on back, crunch up halfway with hands on temples. Extend legs straight out, hover above floor, and flutter/paddle. Add 5 reps per session.
